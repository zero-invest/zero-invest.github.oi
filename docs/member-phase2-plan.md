# 会员系统后续迭代计划

## 当前已完成

- 账户注册、登录、退出、会话 Cookie
- 首次注册赠送 7 天会员
- 会员有效期与流水
- 兑换码使用
- 本地兑换码批量生成脚本
- 人工赞赏订单提交
- 管理端审核通过订单并发放会员
- 邀请奖励：被邀请人 7 天内首单赞赏到账触发
- 游客首页限制与详情页会员拦截

## 第二期建议

### 1. 截图上传能力
- 接入 Cloudflare R2 存储付款截图
- 前端会员中心支持文件上传，不再手填截图 URL
- 存储图片哈希，做重复图拦截

### 2. OCR 审核辅助
- 接入 OCR 服务识别金额、时间、疑似订单号
- 回填到 `reward_orders.ocr_summary`
- 接入 [`/api/admin/order/ocr`](../cloudflare/worker/src/index.js) 形成审核闭环

### 3. 管理审核后台
- 前端新增管理页
- 调用 [`/api/admin/orders`](../cloudflare/worker/src/index.js) 查看待审核订单
- 一键写入 OCR 结果、一键审核通过

### 4. 兑换码导入与批量发售支持
- 将 [`scripts/generate-redeem-codes.mjs`](../scripts/generate-redeem-codes.mjs) 生成的 SQL/CSV 接入正式导入流程
- 增加批次停用、批次查询、批次统计

## 第三期建议

### 1. 自动支付回调
- 若后续使用支持回调和签名验证的支付渠道，可新增回调接口
- 回调后直接自动开通会员
- 自动触发邀请奖励

### 2. 风控增强
- 账号/IP 限流
- 重复截图指纹识别
- 订单号防重复处理
- 异常账户自动标记

### 3. 更完整管理后台
- 会员补单
- 手工赠送时长
- 封禁账户
- 停用兑换码
- 查询会员流水与订单流水
